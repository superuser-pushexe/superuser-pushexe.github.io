Flow Desktop

About Flow Desktop

Flow Desktop is a lightweight, modern desktop environment for Linux, built with XCB for X11 and GTK4 for settings and development tools.

It integrates seamlessly with the Openbox window manager, offering a minimal yet powerful interface.

Perfect for low-resource systems or users who love a clean desktop, Flow Desktop combines performance with modern features.

Features

  • Customizable Taskbar
  • Application Menu
  • System Clock
  • Settings UI
  • Flow Builder
  • Volume Controls
  • Wayland Support

Installation

  1. Clone the repository: git clone https://github.com/superuser-pushexe/Flow-Desktop.git
  2. Set up the build: mkdir build && cd build && meson setup ..
  3. Compile: ninja
  4. Run components: startx ./flow ./flow-settings ./flow-builder

Ensure Openbox is running before starting flow.
