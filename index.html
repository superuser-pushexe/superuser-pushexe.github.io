Flow Desktop

A Lightweight, Modern Linux Desktop

Built with XCB and GTK4 for speed and simplicity.

About Flow Desktop

Flow Desktop is a lightweight, modern desktop environment for Linux, built with XCB for X11 and GTK4 for settings and development tools. It integrates seamlessly with the Openbox window manager, offering a minimal yet powerful interface.

Perfect for low-resource systems or users who love a clean desktop, Flow Desktop combines performance with modern features.

Features

  • Customizable Taskbar
  • Application Menu
  • System Clock
  • Settings UI
  • Flow Builder
  • Volume Controls
  • Wayland Support

Installation

  1. Clone the repository: git clone https://github.com/superuser-pushexe/Flow-Desktop.git
  2. Set up the build: mkdir build && cd build && meson setup ..
  3. Compile: ninja
  4. Run components: startx ./flow ./flow-settings ./flow-builder

Ensure Openbox is running before starting flow.

Usage

  • Taskbar (flow) - Displays apps, volume controls, and a clock.
  • Settings (flow-settings) - Change wallpapers and system info.
  • Builder (flow-builder) - Code editor and terminal integration.

Flow Desktop: An Open-Source Linux Desktop Environment

Flow Desktop is a lightweight Linux desktop built with XCB for X11 and GTK4 for settings. Designed for performance, simplicity, and customization, it delivers an elegant experience that seamlessly integrates with Openbox. This project emphasizes minimal resource consumption without sacrificing powerful features.

At its core, Flow Desktop produces a modern user interface complete with customizable elements. The built-in taskbar displays active applications, offers quick access to system notifications, and even provides intuitive volume controls. The Application Menu is neatly organized, making it easy to access all installed programs while the System Clock remains elegantly integrated for time management. With the Settings UI powered by GTK4, users can effortlessly modify wallpapers, configure display settings, and personalize the entire desktop experience.

One of the standout features of Flow Desktop is its Flow Builder—an integrated development environment that includes terminal support, enabling users to code and test applications in a seamless manner. Additionally, Flow Desktop supports Wayland, making it adaptable to modern Linux display protocols, while also ensuring that multi-monitor setups receive specialized attention through enhanced configurations.

The project is hosted on GitHub, allowing developers and enthusiasts alike to contribute. Whether you're interested in submitting bug fixes, suggesting new features, or participating in discussions, Flow Desktop welcomes collaboration. Every pull request, issue report, and piece of documentation added helps improve the experience for all users. The repository is actively maintained, and future improvements aim to incorporate more dynamic widgets, enhanced multi-monitor support, and even smoother transitions for an ultra-modern look.

If you're passionate about open-source, we invite you to visit the repository, clone the code, and join our collaborative journey. Through meaningful contributions and community insights, Flow Desktop will continue to evolve and push the boundaries of what a lightweight Linux desktop can offer.
