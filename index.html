About Flow Desktop

Welcome to Flow Desktop, a lightweight and highly customizable desktop environment designed to bring a modern, seamless user experience to your system. Built with Python, PyQt5, and Xlib, Flow Desktop is inspired by the elegance and functionality of KDE Plasma, but with a focus on minimal resource usage and maximum flexibility.

Flow Desktop is perfect for users who want a clean, efficient, and visually appealing desktop environment that can be tailored to their needs. Whether you're a developer looking for a distraction-free workspace, a power user who needs quick access to applications, or someone who simply wants a beautiful desktop, Flow Desktop has something to offer.

Why Use Flow Desktop?

  • Lightweight Performance: Flow Desktop is designed to run smoothly on a wide range of hardware, from low-end machines to high-performance systems.
  • Customization: With the built-in settings app, you can easily change wallpapers, adjust the taskbar position, customize theme colors, and manage your favorite apps.
  • Modern Features: Enjoy desktop icons with drag-and-drop, a taskbar with an app launcher and system tray, and window management features like snapping and keyboard shortcuts.
  • Cross-Platform Support: While primarily designed for Linux, Flow Desktop includes wallpaper support for Windows and macOS, with plans for broader compatibility in the future.
  • Open Source: Flow Desktop is open-source, meaning you can contribute to its development, customize it to your liking, or even fork it for your own projects.

Whether you're setting up a new system, looking to replace your current desktop environment, or experimenting with a custom setup, Flow Desktop provides a fresh and intuitive experience that adapts to your workflow.

Project Overview

Flow Desktop is a lightweight, customizable desktop environment built with Python, PyQt5, and Xlib. It provides a modern desktop experience with features like desktop icons, a taskbar with an app launcher, a system tray, window management, and a settings app. This documentation details the project structure, each file's purpose, and the functions within them.

Flow Desktop aims to replicate the functionality of a modern desktop environment like KDE Plasma while remaining lightweight and modular. Key features include:

  • Desktop Icons: View and interact with files/folders on the desktop with drag-and-drop and context menus.
  • Taskbar: Includes an app launcher with search and icons, window previews, and a system tray showing clock, battery, and network status.
  • Window Management: Supports window dragging, snapping (left/right), minimize (Alt+M), and maximize/restore (Alt+F).
  • Settings App: Customize wallpaper, taskbar position, theme colors, and manage taskbar apps.
  • Cross-Platform Wallpaper: Wallpaper support for Linux, Windows, and macOS.

Project Structure

The project consists of the following files:

flow/
├── config.json        # Configuration file for wallpaper, apps, and settings
├── dark.qss          # Stylesheet for dark theme
├── light.qss         # Stylesheet for light theme
├── logo.png          # Logo image for the taskbar
├── main.py           # Entry point to start the desktop
├── settings.py       # Settings app for customizing the desktop
├── taskbar.py        # Taskbar with app launcher and system tray
├── tray.py           # System tray component showing clock, battery, and network
├── wallpaper.py      # Wallpaper management functions
├── wm.py             # Window manager for handling window events
├── desktop.py        # Desktop component for managing icons
└── README.md         # Project documentation and usage instructions

Dependencies

  • Python 3.x
  • PyQt5: For GUI components (pip install PyQt5)
  • python-xlib: For window management on Linux (pip install python-xlib)
  • psutil: For system information in the tray (pip install psutil)

File-by-File Documentation

1. config.json

This JSON file stores the configuration for the desktop environment.

Structure:

{
  "wallpaper": "wallpaper_lake.png",
  "taskbar_position": "top",
  "theme": "light",
  "theme_color": "#222222",
  "apps": [
    { "name": "Terminal", "command": ["x-terminal-emulator"], "icon": "/usr/share/icons/terminal.png" },
    { "name": "File Manager", "command": ["xdg-open", "."], "icon": "/usr/share/icons/folder.png" },
    { "name": "Browser", "command": ["xdg-open", "http://www.google.com"], "icon": "/usr/share/icons/browser.png" },
    { "name": "VS Code", "command": ["code"], "icon": "/usr/share/icons/vscode.png" },
    { "name": "Calculator", "command": ["gnome-calculator"], "icon": "/usr/share/icons/calculator.png" }
  ]
}

Fields:

  • wallpaper: Path to the wallpaper image.
  • taskbar_position: "top" or "bottom".
  • theme: "light" or "dark".
  • theme_color: Hex color code for the theme.
  • apps: List of apps in the taskbar, each with name, command, and optional icon.

Usage Instructions

Follow these steps to set up and use Flow Desktop on your system:

1. Installation

  1. Download .deb installer: Download the latest version of Flow Desktop from the Downloads section.
  2. Install the files: Install the .deb package with a package manager UI like synaptic or gDebi. Alternatively, you could manually install it with sudo dpkg -i /path/to/flow-desktop-x11_1.0.2_all.deb
  3. Install Dependencies: Your PC should automatically install the required dependencies, but if it fails to, install them with: 
    pip install PyQt5 python-xlib psutil

2. Running Flow Desktop

  1. Start Flow Desktop: In the terminal, navigate to the project directory and run: 
    python3 main.py
    This will launch the desktop environment, including the wallpaper, taskbar, desktop icons, and window manager.
  2. Access the Settings App: The settings app should open automatically. If not, you can restart Flow Desktop, and it will appear. Use it to customize your wallpaper, taskbar position, theme, and apps.
  3. Interact with the Desktop:
    • Click the logo on the taskbar to open the app launcher and search for apps.
    • Right-click on desktop icons to open or delete files/folders.
    • Right-click on the desktop to create a new folder.
    • Drag windows to snap them to the left or right, or use Alt+M to minimize and Alt+F to maximize/restore.

3. Customizing Flow Desktop

Use the settings app to:

  • Change the wallpaper by selecting a new image file.
  • Add or remove apps from the taskbar by entering their name and command.
  • Adjust the taskbar position (top or bottom) and theme color.
Note: Some changes, like taskbar position, require restarting Flow Desktop to take effect.

4. Stopping Flow Desktop

To stop Flow Desktop, close the terminal window where main.py is running, or press Ctrl+C in the terminal. This will terminate all components.

Limitations

While Flow Desktop offers a modern and lightweight desktop experience, it has some limitations that users should be aware of:

  • Linux-Centric Window Management: The window manager (wm.py) relies on python-xlib, which is only available on Linux. As a result, window management features (e.g., snapping, minimize/maximize) are not supported on Windows or macOS.
  • Limited System Tray Functionality: The system tray currently displays basic information (clock, battery, network status), but it lacks interactive features like volume control or network settings.
  • No Multi-Monitor Support: Flow Desktop does not yet support multiple monitors. The taskbar and desktop icons are only displayed on the primary screen.
  • Basic Theme Support: While light and dark themes are available, the theming system is limited to basic color changes. Advanced theming (e.g., custom widget styles, transparency effects) is not yet implemented.
  • Performance on Low-End Systems: Although designed to be lightweight, Flow Desktop may experience performance issues on extremely low-end hardware due to the use of PyQt5 for GUI rendering.
  • No Built-In File Manager: Desktop icons allow basic interaction (open, delete), but there is no integrated file manager for advanced file operations.
  • Limited Error Handling: Some components may not handle errors gracefully (e.g., missing icons, invalid commands in config.json), which could cause crashes.

We are actively working on addressing these limitations in future updates. Contributions from the community are welcome to help improve Flow Desktop!

Contributing to Flow Desktop

Flow Desktop is an open-source project, and we welcome contributions from the community! Whether you're a developer, designer, or user with ideas, there are many ways to get involved.

How to Contribute

  1. Fork the Repository: Start by forking the Flow Desktop repository on GitHub (link to be added once the project is hosted).
  2. Clone the Repository: Clone your forked repository to your local machine: 
    git clone https://github.com/your-username/flow-desktop.git
  3. Create a Branch: Create a new branch for your changes: 
    git checkout -b feature/your-feature-name
  4. Make Changes: Implement your changes or fixes. Be sure to follow the project structure and coding style (e.g., use consistent indentation, add comments where necessary).
  5. Test Your Changes: Test your changes locally by running Flow Desktop (python3 main.py) to ensure everything works as expected.
  6. Commit Your Changes: Commit your changes with a descriptive message: 
    git commit -m "Add feature: your feature description"
  7. Push to Your Fork: Push your changes to your forked repository: 
    git push origin feature/your-feature-name
  8. Create a Pull Request: Go to the original Flow Desktop repository on GitHub and create a pull request from your branch. Provide a detailed description of your changes and why they are beneficial.

Contribution Ideas

Here are some areas where contributions would be especially helpful:

  • Cross-Platform Window Management: Extend the window manager to support Windows and macOS, possibly by integrating alternative libraries.
  • Enhanced System Tray: Add interactive features to the system tray, such as volume control, network settings, or notifications.
  • Multi-Monitor Support: Implement support for multiple monitors, ensuring the taskbar and desktop icons work across all screens.
  • Advanced Theming: Improve the theming system to support custom widget styles, transparency, and animations.
  • Bug Fixes: Address known issues, such as error handling for missing icons or invalid commands in config.json.
  • Documentation Improvements: Enhance this documentation with more examples, screenshots, or video tutorials.
