A Lightweight X11 Desktop Environment Built for Speed and Simplicity

Get Started

About Flow Desktop

Flow Desktop is a lightweight desktop environment built on X11, written in C, and designed to be fast, minimal, and easily customizable. Using the Openbox window manager, it provides a simple taskbar with application launchers and a clock, optimized for low-resource usage while retaining essential desktop functionality.

Ideal for users who need a no-frills, efficient desktop on older hardware or minimal systems, Flow Desktop combines performance with simplicity. It’s perfect for developers, tinkerers, or anyone who values a clean workspace without unnecessary overhead.

Why Choose Flow Desktop?

  • Ultra-Lightweight: Minimal resource usage for older or low-spec hardware.
  • Customizable: Easily tweak the taskbar and launchers by editing the source code.
  • Fast: Built in C with X11 for rapid performance.
  • Open Source: Released under the MIT License—modify and share freely.

Installation

Building from Source

  1. Clone the repository: 
    git clone https://github.com/superuser-pushexe/Flow-Desktop.git
cd Flow-Desktop
  2. Compile the code: 
    gcc -o flow-desktop flow-desktop.c -lX11
  3. Run the desktop: 
    ./flow-desktop
    Or start with X: 
    startx ./flow-desktop

Using Pre-built Binary

  1. Download the latest release from the Releases page.
  2. Make it executable: 
    chmod +x flow-desktop
  3. Run it: 
    ./flow-desktop
    Or with X: 
    startx ./flow-desktop

Project Overview

Flow Desktop is a lightweight X11 desktop environment written in C, utilizing Openbox as its window manager. It features a simple taskbar with up to 5 application launchers and a real-time clock, designed for minimal resource usage and compatibility with other window managers via an override-redirect taskbar.

Key features include:

  • Configurable taskbar height and launcher layout.
  • Customizable application launchers (e.g., terminal, browser, file manager).
  • Live system clock display.
  • Basic X11 error handling for stability.

Project Structure

The project is minimal and consists of the following:

Dependencies

  • X11 Development Libraries (libx11-dev on Debian/Ubuntu)
  • Openbox Window Manager
  • GCC Compiler
  • Basic X11 Utilities (e.g., xterm recommended)

Install dependencies:

# Debian/Ubuntu
sudo apt-get install libx11-dev openbox xterm pcmanfm

# Fedora
sudo dnf install libX11-devel openbox xterm pcmanfm

Code Details

The core of Flow Desktop is flow-desktop.c, a single-file C program that:

  • Initializes an X11 display and creates a taskbar window.
  • Integrates with Openbox for window management.
  • Draws a taskbar with launcher buttons and a clock using X11 functions.
  • Handles mouse events for launching applications.
  • Updates the clock in real-time using system time.

Key Functions (Assumed)

// Initializes the X11 display and taskbar window
void init_taskbar(Display *display, Window *taskbar);

// Adds a launcher with a label and command
void add_launcher(const char *label, const char *command, int position);

// Updates the clock display
void update_clock(Display *display, Window taskbar);

The code is designed to be lightweight, relying only on X11 and Openbox, with no additional GUI frameworks.

Usage

Once running, Flow Desktop displays a taskbar at the bottom of the screen with:

  • Default Launchers:
    • "Term" - Opens xterm.
    • "Web" - Opens the default browser to Google.
    • "Files" - Opens PCManFM file manager.
  • Clock: Updates in real-time on the right side.

Click the launcher icons to start applications. Use Openbox shortcuts (configured separately) for window management.

Troubleshooting

  • Desktop doesn’t start? Ensure X11 is installed and try startx ./flow-desktop.
  • Launchers not working? Verify the applications (xterm, etc.) are installed.
  • X11 errors? Check terminal output for details.

Customization

Customize Flow Desktop by editing flow-desktop.c before compilation:

  • TASKBAR_HEIGHT: Set taskbar height (default: 30px).
  • LAUNCHER_SIZE: Set launcher icon size (default: 24px).
  • LAUNCHER_PADDING: Adjust spacing between launchers (default: 5px).
  • CLOCK_WIDTH: Reserve space for the clock (default: 100px).

To modify launchers, edit the add_launcher() calls in main():

add_launcher("Term", "xterm", 0);
add_launcher("Web", "firefox", 1);

Recompile after changes with gcc -o flow-desktop flow-desktop.c -lX11.
